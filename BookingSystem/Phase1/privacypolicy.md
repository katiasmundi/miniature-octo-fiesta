# Privacy Policy

This Privacy Policy describes how personal data is collected, used, and protected in the Booking System application.

## 1. Data Controller
The data controller for this system is the organization operating the Booking System.  
This application is used for educational and testing purposes.

## 2. Personal Data Collected
The system collects and processes the following personal data:

- Email address (used as username)
- Password (stored securely as a hash)
- Date of birth (used to verify minimum age requirement)
- User role (reserver or administrator)
- Reservation data (resource, time period)
- Technical identifiers (user token)

No unnecessary personal data is collected.

## 3. Purpose of Data Processing
Personal data is processed for the following purposes:

- User authentication and authorization
- Managing reservations and resources
- Enforcing role-based access control
- Ensuring system security and accountability

## 4. Legal Basis for Processing
Personal data is processed based on user consent given during registration and the legitimate interest of operating the booking system.

## 5. Data Minimization and Storage
Only the minimum amount of personal data required for system functionality is stored.  
Passwords are never stored in plain text and are hashed using a strong cryptographic algorithm.

Personal data is stored only for as long as the user account exists or as required for system operation.

## 6. User Rights
Users have the following rights under GDPR:

- The right to access their personal data
- The right to request correction of inaccurate data
- The right to request deletion of their account ("right to be forgotten")

Account deletion is performed by an administrator upon request.

## 7. Data Security
Appropriate technical and organizational security measures are implemented, including:

- Secure password hashing
- Role-based access control
- Protection against common web vulnerabilities (e.g. CSRF, XSS, SQL Injection)

## 8. Data Sharing
Personal data is not shared with third parties.  
This system does not use external analytics or tracking services.

## 9. Changes to This Policy
This privacy policy may be updated if the system functionality changes.  
Users will be informed of significant changes.

## 10. Contact Information
For privacy-related questions or requests, please contact the system administrator.
